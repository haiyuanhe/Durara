搭建你的个人博客

1.环境

  • 你需要有一台已经安装好的git 和node 环境的电脑.
  • git 能够通过ssh 直接连上你的github
  • node安装完成之后需要安装npm 能够允许npm 命令.

2.安装hexo

  1. 一句话安装 npm install -g hexo-cli
  2. 然后在你的想建站的目录上面 hexo init
  3. _config.yml 根据你的需要修改这里的配置文件
  4. 最后只需要通过 hexo server 命令就能够直接通过本地访问 localhost:4000
  5. 具体可以参考hexo使用说明
  6. 将你的hexo根目录deploy 到远程的github 上面.首先完成下面的
    • ps: 需要先创建一个仓库
    • 比如 我的账号名:haiyuanhe 创建仓库: haiyuanhe.github.io
    • 这样就能够通过haiyuanhe.github.io 直接访问到你的目录下面的index.html
  7. deploy 也需要配置_config.yml,可以参考 hexo git部署
  8. 配置完成之后, 你只要有任何修改,每次通过三个命令

然后就能将你的新加入的东西或者修改的东西放上github 上面. 这个时候就可以通过
.github.io 访问到你的网站.

3. 安装主题

先挑选你喜欢的主题.主要参考地址

官方上的主题

github 上自己找
看到你想要的之后,只需要在你的那个hexo 根目录下面 clone 你想要的皮肤.
